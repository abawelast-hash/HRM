هذا الملف يشرح كل متغير وكل دالة وكل كلاس وكل خاصية في المشروع بالتفصيل، مُرتباً حسب الملفات.

## 1. src/utils/config.py — مُحمِّل الإعدادات

### المتغيرات العامة (Module-level)

| المتغير | النوع | الوصف |
|---------|-------|-------|
| `_CONFIG_DIR` | `Path` | المسار المطلق لمجلد `config/`. يُحسب تلقائياً من موقع الملف الحالي بالصعود 3 مستويات |

### الدوال

#### `_resolve_env_vars(value)`
- **المُدخل**: `value` — أي قيمة (نص، قاموس، قائمة، أو غيرها)
- **المُخرج**: نفس القيمة لكن مع استبدال أنماط `${VAR}` بقيم متغيرات البيئة
- **الشرح**: تعمل بشكل عودي (Recursive). إذا كانت القيمة نصاً تبحث عن `${...}` وتستبدلها. إذا كانت قاموساً تُعالج كل قيمة. إذا كانت قائمة تُعالج كل عنصر
- **مثال**: `"${POSTGRES_URL}"` ← يصبح ← `"postgresql://user:pass@localhost:5432/db"`

#### `load_config(filename: str) -> dict`
- **المُدخل**: `filename` — اسم ملف YAML (مثل `"settings.yaml"`)
- **المُخرج**: قاموس Python بمحتوى الملف بعد استبدال متغيرات البيئة
- **الشرح**: يفتح الملف من `_CONFIG_DIR`، يقرأه بـ `yaml.safe_load`، ثم يُمرره عبر `_resolve_env_vars`

#### `get_settings() -> dict`
- يُرجع محتوى `config/settings.yaml`

#### `get_sources() -> dict`
- يُرجع محتوى `config/sources.yaml`

#### `get_model_config() -> dict`
- يُرجع محتوى `config/model_config.yaml`

#### `get_alerts_config() -> dict`
- يُرجع محتوى `config/alerts.yaml`

## 3. src/utils/batch_size_estimator.py — مُقدِّر حجم الدفعة

### الدوال

#### `estimate_batch_size(model, sample_input: dict, max_batch_size: int = 512, min_batch_size: int = 4, target_memory_fraction: float = 0.7) -> int`
- **المُدخلات**:
  - `model` — نموذج PyTorch
  - `sample_input` — قاموس يحتوي عينة إدخال واحدة (tensors)
  - `max_batch_size` — الحد الأقصى للحجم (افتراضي: 512)
  - `min_batch_size` — الحد الأدنى (افتراضي: 4)
  - `target_memory_fraction` — نسبة ذاكرة GPU المستهدفة (افتراضي: 0.7 = 70%)
- **المُخرج**: `int` — أكبر حجم دفعة يناسب ذاكرة GPU
- **الشرح**: إذا لم يتوفر GPU، يُرجع `min_batch_size`. وإلا يختبر أحجاماً متزايدة (4, 8, 16, 32...) حتى يفشل أو يتجاوز حد الذاكرة
- **الخوارزمية**: بحث ثنائي — يُضاعف الحجم في كل خطوة، ويتوقف عند RuntimeError (نفاد الذاكرة)

## 4. src/utils/validation_utils.py — أدوات التحقق

### الدوال

#### `validate_dataframe(df: pd.DataFrame, required_columns: list, name: str = "DataFrame")`
- **المُدخلات**: DataFrame وقائمة أعمدة مطلوبة
- **المُخرج**: `True` أو يُلقي `ValueError`
- **الشرح**: يتحقق من وجود كل الأعمدة المطلوبة وأن DataFrame ليس فارغاً

#### `check_nan_ratio(df: pd.DataFrame, threshold: float = 0.5) -> dict`
- **المُدخلات**: DataFrame وعتبة (افتراضي 50%)
- **المُخرج**: قاموس الأعمدة التي تتجاوز نسبة NaN فيها العتبة
- **مثال**: `{"price": 0.65}` يعني أن عمود price فيه 65% قيم مفقودة

#### `validate_price_range(price: float, min_val: float = 100, max_val: float = 1_000_000) -> bool`
- **الشرح**: يتحقق أن سعر الليرة السورية ضمن نطاق معقول (100 إلى مليون)

#### `detect_outliers_iqr(series: pd.Series, factor: float = 1.5) -> pd.Series`
- **المُدخلات**: سلسلة بيانات ومُعامل IQR (افتراضي 1.5)
- **المُخرج**: قناع بولياني (True = قيمة شاذة)
- **الخوارزمية**: IQR Method — الشاذ هو أي قيمة خارج `[Q1 - 1.5×IQR, Q3 + 1.5×IQR]`

#### `validate_time_continuity(timestamps: pd.Series, expected_freq: str = "H", max_gap_hours: int = 72) -> list`
- **المُدخلات**: سلسلة طوابع زمنية وأقصى فجوة مسموحة (72 ساعة)
- **المُخرج**: قائمة tuple تحتوي (بداية الفجوة، نهايتها، مدتها)
- **الشرح**: يكشف الفجوات الزمنية الكبيرة التي قد تحتاج استيفاء

## 5. src/ingestion/collectors/base.py — الكلاس الأساسي للجامعات

### كلاس `BaseCollector(ABC)`

هذا كلاس مجرد (Abstract) — لا يمكن إنشاء كائن منه مباشرة. يجب إنشاء كلاس يرثه ويُنفذ دالة `collect()`.

#### الخصائص (Attributes)

| الخاصية | النوع | الوصف |
|---------|-------|-------|
| `self.name` | `str` | اسم الجامع (مثل `"enab_baladi"`) |
| `self.source_type` | `str` | نوع المصدر: `"rss"`, `"telegram"`, `"web"`, `"api"` |
| `self.session` | `None` | جلسة HTTP (تُستخدم عند الحاجة) |

#### الدوال

##### `__init__(self, name: str, source_type: str)`
- مُنشئ الكلاس. يُعيِّن الاسم والنوع

##### `collect(self, **kwargs) -> List[Dict[str, Any]]` *(مجرد - Abstract)*
- **يجب تنفيذه في الكلاسات الفرعية**
- يجمع البيانات من المصدر ويُرجع قائمة قواميس

##### `compute_hash(self, content: str) -> str`
- **المُدخل**: نص `content`
- **المُخرج**: hash بتشفير SHA-256 (نص سداسي عشري بطول 64 حرف)
- **الاستخدام**: منع تكرار البيانات — إذا تكرر الـ hash، البيانات مكررة

##### `to_unified_format(self, raw_data: Dict) -> Dict`
- **المُدخل**: قاموس بيانات خام من المصدر
- **المُخرج**: قاموس بالتنسيق الموحد:
  ```python
  {
      "id": "uuid4...",           # معرف فريد
      "source": "enab_baladi",    # اسم المصدر
      "source_type": "rss",       # نوع المصدر
      "timestamp": datetime(...),  # وقت البيانات
      "raw_text": "...",          # النص الخام
      "raw_numeric": 14500.0,     # القيمة الرقمية (اختياري)
      "language": "ar",           # اللغة
      "metadata": {...},          # بيانات إضافية
      "content_hash": "sha256..." # بصمة المحتوى
  }
  ```

##### `collect_with_retry(self, max_retries: int = 3, **kwargs) -> List[Dict[str, Any]]`
- يستدعي `collect()` مع إعادة المحاولة عند الفشل
- **التأخير**: تصاعدي أسي — 1 ثانية، 2 ثانية، 4 ثوان
- بعد استنفاد المحاولات، يُلقي Exception

## 7. src/ingestion/collectors/telegram_collector.py — جامع تلغرام

### كلاس `TelegramCollector(BaseCollector)`

#### الخصائص

| الخاصية | النوع | الوصف |
|---------|-------|-------|
| `self.channel` | `str` | اسم القناة (مثل `"@syrian_exchange_rates"`) |
| `self.clients` | `List[TelegramClient]` | قائمة عملاء تلغرام (حسابات متعددة) |
| `self.current_client_idx` | `int` | فهرس العميل الحالي (للتدوير) |

#### الدوال

##### `__init__(self, name: str, channel: str, api_credentials: List[Dict])`
- يُنشئ `TelegramClient` لكل مجموعة بيانات اعتماد

- تدعم استخدام `async with` — تبدأ وتُنهي جميع الجلسات

##### `get_next_client(self) -> TelegramClient`
- يُرجع العميل الحالي ويُنقل المؤشر للعميل التالي (Round-Robin)

##### `collect(self, limit: int = 100, since_hours: int = 24) -> List[Dict]`
- يجمع آخر `limit` رسالة من القناة خلال آخر `since_hours` ساعة
- يتخطى الرسائل بدون نص
- يضيف تأخير عشوائي (0.5-2.0 ثانية) بين الرسائل لتجنب الحظر
- عند FloodWaitError: ينتظر المدة المطلوبة ثم يعيد المحاولة بعميل آخر

## 12. src/processing/text/cleaner.py — مُنظِّف النصوص

### كلاس `TextCleaner`

#### المتغيرات الثابتة (Class-level)

| المتغير | النوع | الوصف |
|---------|-------|-------|
| `_NORM_MAP` | `dict` (ترجمة) | خريطة تطبيع الأحرف: أ/إ/آ→ا، ة→ه، ى→ي |
| `_NUMERAL_MAP` | `dict` (ترجمة) | خريطة تحويل الأرقام الشرقية (٠١٢...) إلى الغربية (012...) |

#### الدوال

##### `clean(self, text: str) -> str`
- خط الأنابيب الكامل — يستدعي 8 دوال بالترتيب:
  1. `remove_html()` — إزالة وسوم HTML
  2. `remove_urls()` — إزالة الروابط
  3. `remove_mentions_hashtags()` — إزالة @ و #
  4. `remove_emojis()` — إزالة الرموز التعبيرية
  5. `normalize_arabic()` — تطبيع الأحرف
  6. `convert_numerals()` — تحويل الأرقام
  7. `remove_diacritics()` — إزالة التشكيل (الفتحة والضمة والكسرة...)
  8. `normalize_whitespace()` — توحيد المسافات

##### `remove_html(text)` / `remove_urls(text)` / `remove_mentions_hashtags(text)` / `remove_emojis(text)`
- جميعها ثابتة (Static) تستخدم `re.sub()` لإزالة الأنماط المُحددة

##### `normalize_arabic(self, text)` / `convert_numerals(self, text)`
- تستخدم `str.translate()` مع خرائط الترجمة

##### `remove_diacritics(text)` *(ثابت)*
- يزيل الحركات العربية (Unicode range: `\u0617-\u061A` و `\u064B-\u0652`)

##### `normalize_whitespace(text)` *(ثابت)*
- يستبدل عدة مسافات متتالية بمسافة واحدة

## 14. src/processing/numeric/extractor.py — مُستخرج الأرقام

### كلاس `NumericExtractor`

#### المتغيرات الثابتة

| المتغير | النوع | الوصف |
|---------|-------|-------|
| `_PRICE_PATTERNS` | `list[str]` | 3 أنماط regex لاستخراج الأسعار: (1) رقم + ل.س/ليرة/SYP، (2) الدولار + رقم، (3) أي رقم من 4-7 خانات |
| `_PCT_PATTERN` | `re.Pattern` | نمط لاستخراج النسب المئوية: `(\d+\.?\d*)\s*%` |
| `_UP_WORDS` | `set` | كلمات الارتفاع: ارتفع، ارتفاع، صعود، زيادة، ارتقى، صاعد |
| `_DOWN_WORDS` | `set` | كلمات الانخفاض: انخفض، انخفاض، هبوط، تراجع، هابط، نزول |
| `_STABLE_WORDS` | `set` | كلمات الاستقرار: استقر، استقرار، ثبات، مستقر |

#### الدوال

##### `extract_price(self, text: str) -> Optional[float]`
- يُجرب أنماط الأسعار الثلاثة بالترتيب
- يتحقق من النطاق (100 - 1,000,000) قبل القبول

##### `extract_percentage(self, text: str) -> Optional[float]`
- يستخرج النسبة ويُحولها لكسر عشري (3.5% → 0.035)

##### `extract_direction(self, text: str) -> Optional[str]`
- يبحث عن كلمات الاتجاه. يُرجع `"up"`, `"down"`, `"stable"`, أو `None`

##### `extract_all(self, text: str) -> dict`
- يستدعي الدوال الثلاث ويُرجع: `{"price": ..., "percentage": ..., "direction": ...}`

##### `validate_price(price, min_val=100, max_val=1_000_000) -> bool` *(ثابت)*
